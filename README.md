# Personal Website

This is the latest version of my personal website.
![A screenshot of the homepage](./homepage.png)

## Development
Running the frontend is simple:
```
cd frontend
npm install
npm run dev
```

## Stack & tools
- React + Vite
- Material icons (from `react-icons`) and Devicons
